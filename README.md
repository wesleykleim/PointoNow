# PontoNow
Uma API para sistemas de pontos digitais.

## Endpoints
- Funcionários
    - Cadastrar
        - Nome 
        - CPF
        - Sexo
        - Cargo
        - Coletar biometria 
        - Reconhecimento Facial
    - Acesso 
         - Login
         - Senha
         - Esqueci a senha
         - Data e hora
    - Identificação 
        - Localização GPS
        - Biometria 
        - Reconhecimento Faical
        - Informar nome da empresa
    - Marcador de Ponto
        - Históriaco de pontos
        - Pontão para validar o ponto 
    - Banco de Horas 
        - Debito de Horas
        - Crédito de horas
        - Agendar Pagamento de Horas
        - Agendar compensação de Horas
